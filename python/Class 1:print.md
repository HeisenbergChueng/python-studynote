## 第一課 print(打印）的使用
1.print顧名思義是打印的意思，即打印框中的內容，例如

```
print(hello world)
```

  即會在結果中顯示 **hello world**

&nbsp;

2.print也可以print出一個指令的內容，例如

```
name = input(你的大名)
print(name)
```
即可在結果中顯示輸入**你的大名：**然後輸入後下一行會出現對應輸入的文字

&nbsp;

3.print可以在括號內輸入\n來換行顯示，例如

```
print（你的名字\n我的姓氏）
```
則結果中顯示為
**你的名字**
**我的姓氏**
